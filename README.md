## How to Use

### Normal Mode
1. Type in the text area to see the heatmap update in real-time
2. Use the sample text buttons to quickly test different content
3. Toggle the spacebar visibility using the sidebar switch

### WPM Test Mode
1. Toggle "WPM mode" in the sidebar
2. Click "Start" to begin the typing test
3. Type the displayed text as quickly and accurately as possible
4. View your WPM score and accuracy when finished
